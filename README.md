# 个人博客系统

## 技术栈

- 前端：Jade + TaildwindCSS + jQuery
- 后端：Express + MySQL

项目使用 Express 生成器生成，包管理工具使用 pnpm（如果本地有，则直接使用）

## 功能

- 前台
    - 文章列表
    - 文章详情
    - 文章分类
    - 文章标签
    - 文章归档
    - 文章搜索
    - 关于
- 后台
    - 文章管理
        - 开源 markdown 编辑器
        - 图片上传
        - 设置分类
        - 设置标签
        - 上传附件
        - 草稿 & 发布 & 隐藏
    - 分类管理（增删查改）
    - 标签管理（增删查改）
    - 附件管理（增删查改）
    - 备份恢复
    - 登录
- Docker 部署
- 环境变量配置
- 日志打印

要求有清晰的目录管理，封装一些公共工具方法，如错误处理，认证中间件等。

先设计好数据库。

页面要体现简洁的风格。
